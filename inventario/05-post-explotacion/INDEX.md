# Fase 05: Post-Explotación

La post-explotación abarca las acciones realizadas tras obtener acceso inicial, con el objetivo de elevar privilegios, asegurar la persistencia y expandir el control hacia otros sistemas de la red.

## Categorías

### [Exfiltración](./exfiltracion/INDEX.md)
Técnicas para extraer datos del sistema comprometido de forma segura y sigilosa.

### [Movimiento Lateral](./lateral-movement/INDEX.md)
Navegación por la red interna para comprometer otros hosts y servicios.

### [Persistencia](./persistencia/INDEX.md)
Mecanismos para mantener el acceso al sistema a pesar de reinicios o cambios de credenciales.

### [Escalada de Privilegios](./privilege-escalation/INDEX.md)
Elevación de permisos desde un usuario limitado a root (Linux) o SYSTEM (Windows).
