# Fase 02: Enumeración

La enumeración es el proceso de establecer una conexión activa con el objetivo y realizar consultas directas para obtener información detallada sobre la infraestructura.

## Categorías

### [Fuzzing](./fuzzing/INDEX.md)
Descubrimiento de recursos ocultos (archivos, directorios, subdominios) mediante fuerza bruta dirigida.

### [Red](./red/INDEX.md)
Análisis de protocolos de infraestructura (DNS, SNMP, SMB, etc.).

### [Servicios](./servicios/INDEX.md)
Enumeración detallada de aplicaciones y bases de datos específicas.

### [Web](./web/INDEX.md)
Análisis de la superficie de ataque HTTP/HTTPS.
