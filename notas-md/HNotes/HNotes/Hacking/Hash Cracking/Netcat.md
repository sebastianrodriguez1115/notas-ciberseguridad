# Netcat

Netcat, also known as "nc", is a versatile networking utility that allows for reading from and writing to network connections using TCP or UDP protocols. It can be used for a variety of purposes, such as port scanning, file transfers, and debugging network issues. Netcat provides a command-line interface with various options and modes, making it a powerful tool for network administrators and security professionals.

You can find more information about Netcat [here](Netcat%20938d2a3c6c7a467ca03a74b3f2a99b14.html).

```
nc -nvlp 1234
```
