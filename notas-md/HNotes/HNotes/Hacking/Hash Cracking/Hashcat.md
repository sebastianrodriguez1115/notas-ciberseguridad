# Hashcat

Hashcat is a powerful password recovery tool used to crack hashes. It supports various types of hash algorithms and can be used for both offline and online password cracking. Hashcat is widely used by security professionals and ethical hackers to test the strength of passwords and improve security measures.

```
hashcat -a 0 -m 1600 hash.txt /usr/share/wordlists/rockyou.txt
```
