# Notas de Estudio en Ciberseguridad

Este repositorio reúne mis notas de estudio de ciberseguridad. Lo uso como una base de conocimiento personal para organizar técnicas, herramientas, comandos, contramedidas y referencias mientras avanzo en pentesting, seguridad web, fundamentos, post-explotación y DFIR.

El contenido no pretende ser una guía definitiva ni documentación oficial. Es un inventario vivo, escrito para estudiar, repasar y conectar conceptos de forma práctica.

He usado IA como apoyo para construir, ordenar y revisar el inventario. La estructura del repositorio también está pensada para que los agentes puedan leerlo eficientemente: índices jerárquicos, metadatos consistentes, slugs estables y archivos Markdown pequeños por técnica.

## Contenido

- [`inventario/`](./inventario/INDEX.md): inventario principal de técnicas, organizado por fases.
- [`learning/`](./learning/): writeups y laboratorios prácticos resueltos durante el estudio.
- [`referencias/`](./referencias/): libros y material de consulta usados como apoyo.
- [`scripts/`](./scripts/): validadores e índices auxiliares para mantener el repositorio consistente.

## Organización

El inventario sigue un flujo cercano al ciclo de un pentest:

1. Reconocimiento
2. Enumeración
3. Análisis de vulnerabilidades
4. Explotación
5. Post-explotación
6. Frameworks y herramientas
7. Fundamentos
8. Forense y DFIR

La entrada principal está en [`inventario/INDEX.md`](./inventario/INDEX.md).

## Nota

Este material está orientado al aprendizaje y a entornos autorizados. Cualquier técnica documentada aquí debe usarse sólo en laboratorios, CTFs, auditorías con permiso explícito o investigación defensiva.
